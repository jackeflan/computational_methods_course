# What input data governs PFAS Plant Uptake?
This project uses a compiled dataset of PFAS plant uptake experiments stored across multiple CSV and Excel files. The data are multivariate and include measured root and shoot concentrations (CR, CS) and derived metrics such as transpiration stream concentration factors (TSCF), along with associated chemical and plant parameters. Input variables include PFAS properties such as chain length and functional group, partitioning coefficients (Kd, KRX), and plant-specific parameters such as transpiration rate (Q), root and shoot mass (MR, MS), and kinetic constants (kR, kS). The dataset requires preprocessing to merge across files, standardize units (e.g., dry weight vs fresh weight), and handle missing values. In addition to direct statistical analysis, I will perform model calibration by fitting parameters (e.g., KRX and R) using an ODE plant uptake model. This allows for the dataset to be transformed from observations into physical parameters. The primary computational method will be principal component analysis (PCA) to identify dominant modes of variability in both the original inputs and the calibrated parameters, followed by linear regression to predict uptake metrics (CR, CS, or TSCF) from these variables. Model performance will be evaluated using R^2 and RMSE, with diagnostic plots including predicted versus observed values and PCA variance explained. The calibrated parameters have clear physical interpretations: KRX represents the partitioning between root tissue and xylem water, controlling how strongly PFAS are retained in roots; TSCF describes the efficiency of transport from roots to shoots via the transpiration stream; and R represents a retardation factor relative to water flow, capturing how chemical properties slow or enhance uptake. The goal is to determine whether uptake behavior can be explained by a small number of interpretable physical controls rather than the large number that arrises in plant-soil systems.
